# GenAI Montréal — Vibe Code to Vibe Engineering

Markdown summaries of the three talks from the **GenAI Montréal** meetup, with topics spanning AI-assisted software development, multi-agent systems, and production AI workflows.

The summaries were generated by feeding the recorded audio, slides, and speaker repos into **NotebookLM** and turning everything into clean Markdown. They're hosted here because the write-ups include **Mermaid diagrams**, which GitHub renders natively (and Discord does not).

## Event

- **Meetup:** [Vibe Code to Vibe Engineering](https://www.meetup.com/generative-ai-montreal/events/314705638/) — GenAI Montréal
- **Date:** Thursday, June 18, 2026 · 5:30–7:30 PM EDT
- **Venue:** MIA Innovation @ Maplr — 355 Sainte-Catherine Ouest #500, Montréal, QC

## Presentations

1. **[GOAP Agents with Koog](presentations/goap-agents-with-koog.md)**  
   _MIA Innovation (Québec)._ A real-world quotation agent built with Goal-Oriented Action Planning (GOAP) in Koog, JetBrains' Kotlin/Java agent framework: deterministic A\* planning with AI-driven execution.

2. **[Orchestrated AI Coding](presentations/orchestrated-ai-coding.md)**  
   _Ho Kai, with Faïçal Sawadogo & Mossab Belle._ Moving from messy AI coding to trustworthy AI coding by treating the AI like a team — orchestrator, on-demand skills/agents, gates, and honest stopping.

3. **[Vibe Coding to Vibe Engineering](presentations/vibe-coding-to-vibe-engineering.md)**  
   _A practical Cursor playbook._ Using AI within a disciplined engineering lifecycle — feed the right context, configure with Rules/Commands/Skills/MCP, and match the workflow to the task size.
